# Setup Guide

### 1. Fortinet Requirements
- Enable REST API Admin in System > Administrators.
- Generate an API Token.

### 2. Aruba Requirements
- Ensure the switch is reachable via HTTPS.
- Use a local admin account for the script.

### 3. Local Machine
- Install PowerShell 7.
- Run `Install-Module PowerFGT, PowerArubaCX`.
